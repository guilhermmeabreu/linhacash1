Perguntas Frequentes
O que é o LinhaCash?
Plataforma de análise de dados da NBA para apostadores. Mostramos estatísticas históricas dos jogadores para te ajudar a tomar decisões mais informadas.
Os dados são em tempo real?
Os dados são atualizados diariamente antes dos jogos. Não exibimos estatísticas ao vivo durante as partidas.
Qual a diferença entre Gratuito e Pro?
Gratuito: 1 jogo por dia, 1 jogador por time, apenas PTS. Pro: todos os jogos, todos os jogadores, todas as stats (AST, REB, 3PM, P+A, P+R, A+R, FG2A, FG3A) e períodos H2H, L20 e L30.
Como cancelo minha assinatura?
Acesse Perfil → Suporte → Cancelar assinatura. O acesso Pro permanece ativo até o fim do período já pago.
O LinhaCash é uma casa de apostas?
Não. Somos uma ferramenta de análise de dados. Não aceitamos apostas nem promovemos sites de apostas.

Termos de Uso

1. Finalidade

O LinhaCash é uma plataforma de análise de dados estatísticos da NBA para fins exclusivamente informativos. Não somos uma casa de apostas, não aceitamos apostas e não nos responsabilizamos por decisões financeiras tomadas com base nos dados exibidos.

2. Uso Responsável

As informações têm caráter informativo e estatístico. O usuário é o único responsável por suas decisões. Jogue com responsabilidade. Se tiver problemas com apostas, ligue para o CVV: 188.

3. Planos e Pagamentos

Plano Free: gratuito com acesso limitado. Plano Pro Mensal: R$24,90/mês. Plano Pro Anual: R$197,00/ano (R$16,41/mês). Pagamentos processados pelo Mercado Pago.

4. Política de Reembolso

Por se tratar de serviço digital com acesso imediato, conforme Art. 49 do Código de Defesa do Consumidor (Lei 8.078/90), que exclui serviços de entrega imediata do direito de arrependimento de 7 dias, não realizamos reembolsos após o acesso ser liberado. O cancelamento encerra a renovação automática; o acesso Pro permanece ativo até o fim do período pago.

5. Privacidade e LGPD

Tratamos seus dados conforme a Lei Geral de Proteção de Dados (LGPD — Lei 13.709/2018). Coletamos apenas nome e email. Não vendemos dados. Você pode excluir sua conta e todos os dados a qualquer momento em Perfil → Excluir conta. Política completa: linhacash.com.br/privacidade

6. Precisão dos Dados

Os dados são atualizados diariamente via API oficial. Não nos responsabilizamos por imprecisões ou por perdas financeiras baseadas nos dados exibidos.

7. Propriedade Intelectual

O LinhaCash, suas marcas, design e código são propriedade do operador. É proibida a reprodução sem autorização.

8. Foro

Estes termos são regidos pela legislação brasileira. Fica eleito o foro da comarca de domicílio do operador para dirimir eventuais controvérsias.
