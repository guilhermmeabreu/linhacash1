Perguntas Frequentes
O que é o LinhaCash?
Plataforma de análise de dados da NBA para apostadores. Mostramos estatísticas históricas dos jogadores para te ajudar a tomar decisões mais informadas.
Os dados são em tempo real?
Os dados são atualizados diariamente antes dos jogos. Não exibimos estatísticas ao vivo durante as partidas.
Qual a diferença entre Gratuito e Pro?
Gratuito: 1 jogo por dia, 1 jogador por time, apenas PTS e REB. Pro: todos os jogos, todos os jogadores, todas as stats (AST, 3PM, P+A, P+R) e períodos H2H e L20.
Como cancelo minha assinatura?
Acesse Perfil → Suporte → Cancelar assinatura. O acesso Pro permanece ativo até o fim do período já pago.
O LinhaCash é uma casa de apostas?
Não. Somos uma ferramenta de análise de dados. Não aceitamos apostas nem promovemos sites de apostas.

Termos de Uso

1. Finalidade

O LinhaCash é uma plataforma de análise de dados estatísticos da NBA. Não somos uma casa de apostas.

2. Uso Responsável

As informações são de caráter informativo. O usuário é o único responsável por suas decisões. Aposte com responsabilidade.

3. Privacidade

Seus dados são protegidos e nunca serão vendidos a terceiros.

4. Planos

O plano gratuito oferece acesso limitado. O plano Pro é R$24,90/mês ou R$197,00/ano (equivalente a R$16,41/mês), cancelável a qualquer momento.

5. Política de Reembolso

Não realizamos reembolsos após a confirmação do pagamento, pois o acesso ao conteúdo é liberado imediatamente. Ao assinar, o usuário declara estar ciente e de acordo com esta política. O cancelamento encerra a renovação automática, mas o acesso permanece ativo até o fim do período já pago.

6. Precisão dos Dados

Os dados são atualizados diariamente via API oficial. Não nos responsabilizamos por perdas baseadas nos dados exibidos.
